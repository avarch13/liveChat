## Overview

LiveChat is a real-time messaging application built with React and Spring Boot.

## Features

- Real-time message synchronization
- Clean and intuitive user interface
- Built with modern web technologies

## Tech Stack

- **Frontend**: React with Vite
- **Backend**: Spring Boot
- **Runtime**: Node.js and Java

## Getting Started

### Prerequisites

- Node.js
- Java 11+
- Maven

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/avarch13/liveChat.git
   cd liveChat
   ```

2. Install frontend dependencies
   ```bash
   cd liveChat
   npm install
   ```

3. Build and run the backend
   ```bash
   mvn spring-boot:run
   ```

4. Start the frontend development server
   ```bash
   npm run dev
   ```
